## Features
- MVC architecture for clean separation of concerns
- RESTful routes for application resources
- User-friendly frontend using HTML, CSS, and vanilla JS
- MongoDB database using **Mongoose** ODM
- CRUD operations for core resources
- Form validation and error handling
- Modular code with controllers, models, and views

## Tech Stack
- Node.js
- Express.js
- MongoDB / Mongoose
- HTML5, CSS3, JavaScript
- Nodemon (for development)
- dotenv (environment variables)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/sarah-app.git
cd sarah-app

2. Install dependencies:
npm install

3. Create .env file in root directory:
PORT=3000
MONGO_URI=your_mongodb_connection_string

4.Start the server:
npm run dev
